# API for a remote controlled vacuum cleaner

## Installation

```
npm install
```
## How to run
```
npm run prod

npm run server
```
to launch the frontend open the browser on
```
localhost:5000
```
run tests
```
npm run test-backend
```

You can easily invoke the api via Postman
